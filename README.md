## 💼 Proyecto destacado

### InverZen Landing Page

Landing page profesional para **InverZen Capital Ready IA**, proyecto orientado a diagnóstico financiero para micro y pequeñas empresas.

El objetivo del proyecto es presentar una solución digital que ayude a PYMES a ordenar ventas, costos, gastos, deudas y caja disponible mediante un diagnóstico financiero simple.
